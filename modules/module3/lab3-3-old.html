Lab 3.3 Quantile Regression Modeling#

Why quantile regression?#

What if we think that the rank order of events is basically the same in two datasets, but we don’t know if they’re linearly related

Common example, climate downscaling: Temperatures in a large-scale climate model for the whole state of WA are likely related to temperatures at a point on Mt. Rainier, but maybe not linearly – rather, we think that the hottest/coolest days over all of WA are likely also the hottest/coolest days over a point on Mt. Rainier.

Principal approach:#

  1. For each of your two datasets, create an empirical CDF (similar to what we did before)

# Let's execute step 1:  create 2 empirical CDFs, one for each site.
# For this, we can borrow material from Lab 2.2.  (Note that you could also 
# make the quantile function be an external function.)

# Calculate quantiles of the data

# The function takes an array of data as an argument and returns the data
# as an ordered array and the plotting position associated with each data point.
def quantile_fn(data):

    ordered_data = np.sort(data)
    n = len(ordered_data)

    rank = []
    plotting_position = []
    for i in range(n):
        rank.append(i+1)
        # Using the Cunnane plotting position.
        plotting_position.append((rank[i]-.4)/(n+.2))
        
    return ordered_data, plotting_position

BLC_ordered, BLC_quantile = quantile_fn(data['BLC_max'])
SLI_ordered, SLI_quantile = quantile_fn(data['SLI_max'])

#Let's try plotting these on the same plot and see how they compare.
plt.figure()
plt.plot(BLC_ordered, BLC_quantile, label='Blue Canyon')
plt.plot(SLI_ordered, SLI_quantile, label='Slide Canyon')
plt.ylabel('Quantile')
plt.xlabel('Peak SWE (mm)')
plt.title('Quantiles of SWE data')
plt.legend(loc="best")

So what does “Quantile” mean in the above graph? We just made two empirical CDF’s, or probability of non-exceedance. Note that because these are empirical, these CDF’s do not extend fully to 0 or to 1. If we need to use this to predict extreme values (very high or very low), we would need to linearly interpolate to extend these quantils all the way to 0 and 1. This was illustrated in cell 5 of Lab 2.2. For brevity (and because I don’t need it for this particular example, I won’t repeat that step here). However, note that if you need to use quantile regression on new data (e.g., if you wanted a model to use one snow pillow to predict the other at new times that might be more extreme than your model development period), it is very important that you extend the ends.

Next steps, use these two empirical CDFs as a way of looking-up (or mapping) values from the predictor to the predictand, by matching which physical value corresponds to the same quantile.#

We see that our test value corresponds to the median value at Slide Canyon. (Yes, you would hope so, since I defined it as the median, but it’s always best practice to start coding with a situation where you know the right answer.)

Now, we need to take this Slide Canyon cdf value (0.5) and find the Blue Canyon actual SWE value that corresponds to its same cdf value.

Does it make sense how we mapped from the red line to the blue line (intersection with the Slide Canyone PDF) to the horizontal blue line (quantile of that intersection) to the vertical black line (intersection with the Blue Canyon PDF), which leads us to the Blue Canyon value we want.

# Now, we can do this for every value in the Slide Canyon set to come up with a matching prediction for the Blue Canyon set
f = interp1d(SLI_ordered, SLI_quantile)

g = interp1d(BLC_quantile, BLC_ordered)

BLC_predicted=g(f(data['SLI_max']))

Jessica’s notes on why one should not do statistics in the absense of physically-based science.#

What you see above is that the quantile mapping forces the rank order of the predicted timeseries to match the rank order of the prectictor time series. In this particular case, that isn’t always the best idea. Blue Canyon is at a much lower elevation, and in warmer winters, Blue Canyon will get rain at times when Slide Canyon gets snow. In those years, any attempt at correlation falls apart. Thus, we would want to add a predictor variable that had something about temperature or rain vs snow, which I have not provided you within these examples.
